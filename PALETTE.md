# 🎨 mentask Branding Palette

This document serves as the **Single Source of Truth** for the visual identity of **mentask**.

---

## 🧬 Brand Philosophy: "Modular Thinking"
Mentask is not just a chatbot; it's a **modular cognitive engine**. The design reflects structure, logic, and intelligence.

---

## 🎨 Color Palette

| Color | Hex | Usage |
| :--- | :--- | :--- |
| **Deep Python** | `#1E293B` | Main background, terminal depth |
| **Electric Indigo** | `#6366F1` | Brand primary, highlights, active states |
| **Thinking Gold** | `#FBBF24` | Logic/Thinking blocks, intelligence indicators |
| **Mint Module** | `#10B981` | Success states, connected tools/MCP |
| **Crimson Error** | `#EF4444` | Errors, security warnings |
| **Slate Text** | `#CBD5E1` | Primary readable text |

### ✨ Gradients
- **Primary Flow**: `linear-gradient(135deg, #6366F1 0%, #10B981 100%)`
- **Dark Depth**: `linear-gradient(180deg, #1E293B 0%, #0F172A 100%)`

---

## ⌨️ Typography
- **Primary (Code)**: `JetBrains Mono` or `IBM Plex Mono`
- **Display (Titles)**: `Outfit` or `Inter` (Sans-serif)
- **Fallback**: `monospace`

---

## 🧩 UI Components Reference

### 🛡️ Security Badge
- **Style**: Pill-shaped
- **Color**: `#10B981` (Success)
- **Label**: "TRUSTED"

### 🧠 Thinking Indicator
- **Style**: Pulse animation
- **Color**: `#FBBF24`

---

## ✅ Checklist for New Components
1. [ ] Uses `#6366F1` for primary interaction.
2. [ ] Dark mode background is strictly `#1E293B` or darker.
3. [ ] All icons use the Mint Module green for "connected" state.
4. [ ] Rounded corners are consistent (8px or 12px).
