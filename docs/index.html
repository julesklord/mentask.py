Installation

Python 3.10+ required. Install the latest stable build from PyPI:

# Using pip
$ pip install mentask

# Or from source
$ git clone https://github.com/julesklord/mentask
$ cd mentask && pip install -e .

Pro Tip: On first run, mentask will prompt for your API keys (Gemini, OpenAI, DeepSeek) and store them securely in your system keyring.

Technical Architecture

The system operates across three tightly decoupled layers enforcing strong logical boundaries.

┌─ COGNITIVE LAYER (Thinking)
│  └─ AgentOrchestrator [The Heart]
│     ├─ SessionManager (persistent memory, rollback)
│     ├─ ContextManager (compression, snapping)
│     └─ PluginLoader [The Evolver]
│
├─ TOOL ECOSYSTEM (3-Tier Toolset)
│  ├─ Layer 1: Core Tools (immutable foundation)
│  ├─ Layer 2: MCP Community (modular extensions)
│  └─ Layer 3: Evolved Plugins (agent-forged artifacts)
│
├─ SECURITY & TRUST LAYER
│  ├─ TrustManager (whitelist-based access)
│  └─ SecurityAnalyzer (dangerous pattern detection)
│
└─ LLM BACKEND (Reasoning)
   └─ Gemini / DeepSeek / Claude via models.dev
